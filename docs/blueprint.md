# **App Name**: Acceso Seguro

## Core Features:

- User Authentication: Authenticate users based on username and password.
- Background Display: Display the specified background image (modulo_acceso/Identificacion) within the login panel.
- Form Input Validation: Validate username and password input fields to prevent empty submissions.
- Secure Password Handling: Handle passwords securely during authentication.

## Style Guidelines:

- Primary color: Soft lavender (#D0BFFF), mirroring the imaginative quality of the background image.
- Background color: Very light lavender (#F8F5FF), to subtly blend with the main design while maintaining readability.
- Accent color: Soft pink (#FFB3C6), as a subtle contrast for interactive elements like the 'Entrar' button.
- Body and headline font: 'PT Sans' for a friendly but modern feel, easy readability for input labels and button text.
- Position username input on the left, password input on the right, and a small 'Entrar' button below. Maintain good spacing so that the background image is visible.
- Incorporate simple, flat icons for username and password fields, such as a user icon and a key icon.
